1. Two Sum

题目

Given an array of integers nums and an integer target, return indices of the two numbers such that they add up to target.

You may assume that each input would have exactly one solution, and you may not use the same element twice.

You can return the answer in any order.

例子

Input: nums = [2,7,11,15], target = 9
Output: [0,1]
Explanation: Because nums[0] + nums[1] == 9, we return [0, 1].

Input: nums = [3,2,4], target = 6
Output: [1,2]

Input: nums = [3,3], target = 6
Output: [0,1]

解法一

第一种方式就是最简单的遍历循环，外循环遍历从0到length-2，内层循环从1到length-1.这样所有数都能够相加一遍。这种的时间复杂度比较高，应该达到了O(n2)O(n^2)O(n2)

class Solution:
    def twoSum(self, nums: List[int], target: int) -> List[int]:
        length = len(nums)
        i,j = 0,1
        while i <length-1:
            while j <length:
                if nums[i]+nums[j] == target:
                    return [i,j]
                else:
                    j+=1
            i +=1
            j = i+1

解法二

第二种方法就是先对列表的索引按照列表数值进行从小到大的排序，然后将列表的数值最小值和最大值相加，如果和小于target，那么最小值则改为第二小数值，如果大于target，则最大值改为第二大数值。

此时的时间复杂度应该是排序算法的时间复杂度加上循环算法的时间复杂度，所以应该是O(n×logn+n)O(n\times logn+n)O(n×logn+n)

class Solution:
    def twoSum(self, nums: List[int], target: int) -> List[int]:
        sort_id=sorted(range(len(nums)),key=lambda k:nums[k])#sort the index of the list
        i,j = 0,len(nums)-1
        while i != j:
            if nums[sort_id[i]]+nums[sort_id[j]] == target:
                return [sort_id[i],sort_id[j]]
            elif nums[sort_id[i]] + nums[sort_id[j]] < target:
                i+=1
            else:
                j-=1

解法三

第三种解法是leetcode上说是击败100%的方法，从时间复杂度上看是达到了O(n)O(n)O(n)，但是真正的运行时间比第二种还要长。

  1. 创建一个字典，key值是列表的值，value是列表值所在的索引；

  2. 然后循环判断列表中的素；

  3. 如果target-nums[i]在字典中，则输出当前值的索引和target-nums[i]对应的value值；

  4. 如果target-nums[i]不在字典中，则将nums[i]和索引i加入字典中。

class Solution:
    def twoSum(self, nums: List[int], target: int) -> List[int]:
        idm = {}
        for i in range(len(nums)):
            if target-nums[i] in idm:
                return [i,idm[target-nums[i]]]
            else:
                idm[nums[i]]=i
